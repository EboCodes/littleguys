The Journey

Where they've been &
where they're heading

A fan's breakdown of the project timeline - pulled from what's publicly known.

Phase 01
📣
Community
Build and grow an active holder community across Discord & Twitter
Regular announcements, AMAs, and community calls
Reward early supporters with exclusive access and perks
Collaborative voting on project decisions
Create a sense of belonging around the Little Guys brand
✓ Done
Phase 02
🖼️
NFT
10,000 Little Guys minted on-chain as the core collection
Marketplace listings on OpenSea and leading platforms
NFT staking integrated into browser game (Playside)
Second-gen drops planned for ecosystem expansion
Rarity traits and holder utility embedded into each token
✓ Done
Phase 03
🤝
Partnership
Emmy Award-winning TIME Studios signed to produce the animated series
Collaborations with BAYC, Cool Cats, and CASETiFY
Cross-project activations and co-branded events
Corporate brand deals to grow IP reach
Licensing deals tied to global rollout
✓ Done
Phase 04 - Now
